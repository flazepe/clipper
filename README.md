# clipper

## Example Usages

```
clipper -i input.mp4 -s 2:00-2:30 -s 5:12-5:20 -s 6:11-6:17 output.mp4

clipper -i input.mp4 -s 2:00-2:30 -s 5:12-5:20 -s 6:11-6:17 --fade output.mp4

clipper -i input.mp4 -s 2:00-2:30 -s 5:12-5:20 -s 6:11-6:17 --fade=1 output.mp4

clipper -i input.mp4 -s 2:00-2:30 --cq 16 --hevc output.mp4
```
